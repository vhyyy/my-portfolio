Bài Viết Nổi Bật

Java

1. Class và Object

Class là khuôn mẫu, Object là thực thể cụ thể trong OOP.

class Car { String color = "Red"; }
Car myCar = new Car();
Java

2. Tại sao dùng ArrayList?

ArrayList cho phép thay đổi kích thước mảng linh hoạt.

ArrayList<String> list = new ArrayList<>();
list.add("Java");
Java

3. Xử lý lỗi Try-Catch

Bắt lỗi để chương trình không bị dừng đột ngột.

try { int x = 10/0; } 
catch (Exception e) { ... }
Java

4. Vòng lặp For-each

Duyệt mảng nhanh và gọn gàng hơn.

for (String i : cars) {
  System.out.println(i);
}
JavaScript

5. Var, Let, Const

Ưu tiên dùng `const` và `let` thay cho `var`.

const PI = 3.14;
let age = 20;
JavaScript

6. Arrow Function

Cú pháp hàm ngắn gọn của ES6.

hello = () => "Hello World!";
JavaScript

7. Thao tác DOM

Thay đổi nội dung HTML bằng JS.

document.getElementById("id")
        .innerHTML = "Hi";
JavaScript

8. Template Literals

Nối chuỗi thông minh với dấu backtick.

console.log(`Hi ${name}`);
Network

9. Java Socket

Giao tiếp Client-Server qua mạng.

ServerSocket sv = new ServerSocket(8080);
Socket s = sv.accept();
