Chia Sẻ Kiến Thức

Java

1. Class và Object

Class là khuôn mẫu, Object là thực thể cụ thể. Đây là nền tảng của OOP.

class Car { String color = "Red"; }
Car myCar = new Car();
Java

2. ArrayList vs Array

ArrayList linh hoạt hơn mảng thường vì có thể thay đổi kích thước động.

ArrayList<String> list = new ArrayList<>();
list.add("Java"); // Thêm phần tử
Java

3. Kế thừa (Inheritance)

Dùng từ khóa `extends` để lớp con kế thừa thuộc tính của lớp cha.

class Dog extends Animal {
  void bark() { System.out.println("Gâu"); }
}
Java

4. Interface là gì?

Interface chứa các hàm trừu tượng, giúp đa kế thừa trong Java.

interface Animal { void sound(); }
class Pig implements Animal { ... }
JavaScript

5. Let vs Const vs Var

Nên dùng `const` mặc định, `let` khi cần đổi giá trị. Tránh dùng `var`.

const PI = 3.14; 
let age = 20;    
// var old = 10; (Hạn chế)
JavaScript

6. Arrow Function

Cách viết hàm ngắn gọn hơn trong ES6.

// Cũ: function(a) { return a + 10; }
// Mới:
const add = a => a + 10;
JavaScript

7. Template Literals

Nối chuỗi siêu dễ với dấu backtick (`).

let name = "Huy";
console.log(`Xin chào ${name}!`);
JavaScript

8. DOM Manipulation

Thay đổi nội dung HTML bằng JS.

document.getElementById("demo")
        .innerHTML = "Hello JS";
Python

9. List Comprehension

Tạo list mới từ list cũ chỉ trong 1 dòng code.

nums = [1, 2, 3, 4]
squares = [x**2 for x in nums] 
# Kết quả: [1, 4, 9, 16]
Python

10. Dictionary cơ bản

Lưu trữ dữ liệu dạng Key-Value cực mạnh mẽ.

info = {"name": "Huy", "age": 21}
print(info["name"]) # In ra: Huy
C Language

11. Con trỏ (Pointer)

Biến lưu địa chỉ bộ nhớ của biến khác.

int a = 10;
int *ptr = &a; // ptr lưu địa chỉ của a
printf("%d", *ptr); // In ra 10
Network

12. Socket TCP Client-Server

Mô hình giao tiếp tin cậy qua mạng.

// Server Side (Java Example)
ServerSocket sv = new ServerSocket(8888);
Socket s = sv.accept(); // Chờ kết nối
